    Module-level constants:
    - BASE_URL = "https://mempool.space"
    - RETRY_DELAYS = [1, 2, 4, 8, 16]  (5 attempts)

    fetch_blocks_page(start_height: int) -> list[dict]:
    - Docstring: what it fetches, which endpoint, retry behavior, return type, raises
    - Opens httpx.Client(timeout=30.0) as context manager
    - Iterates over enumerate(RETRY_DELAYS) for retry loop
    - On 429/500/502/503/504: log warning with status, height, attempt, delay; time.sleep(delay); continue
    - On httpx.RequestError: log warning; if not last attempt, time.sleep(delay); continue (do not re-raise mid-loop)
    - On success (any other 2xx): resp.raise_for_status(); return resp.json()
    - After loop exhausted: raise RuntimeError(f"All retries exhausted for height {start_height}")

    Important: time.sleep is inside the retry loop, not after every call. A successful first-attempt call sleeps zero times inside fetch_blocks_page — the 500ms inter-page throttle is applied in the backfill worker (next plan), not here.

    Use logging.getLogger(__name__) for all log output (not print).
